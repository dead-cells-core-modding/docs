# Dead Cells Core Modding

![GitHub License](https://img.shields.io/github/license/dead-cells-core-modding/core)  [![Build And Test](https://github.com/dead-cells-core-modding/core/actions/workflows/build.yml/badge.svg?branch=dev)](https://github.com/dead-cells-core-modding/core/actions/workflows/build.yml)

Dead Cells Core Modding (DCCM) is an unofficial mod framework designed to enable deeper modifications to the game.

:::tip

DCCM currently only supports the **Windows** version on the **Steam** platform.

- You can use **Proton** to run DCCM on **Linux**. ([Installation Guide](./tutorial/install-workshop.md))

:::

:::warning

DCCM remains under active development. This means:

- The API may undergo significant changes without prior notice.
- Confusing bugs may occur.
  - If you encounter bugs, please [report them to us](#reporting-issues).

:::

## Translation Documentation

Currently, DCCM documentation is not available in all languages. If you find that your desired language is not provided, you can refer to the [Docusaurus Documentation](https://docusaurus.io/docs/i18n/git) to perform translation work.

## Reporting Issues

:::warning
As the developer is a high school senior with limited maintenance time, bug fixes may take considerable time. Your understanding is appreciated.
:::

:::tip
Before submitting, please verify the issue hasn't been reported previously.
:::

Create a new issue on the [Github Issues](https://github.com/dead-cells-core-modding/core/issues/new) page.

Your issue should include:

- **Problem Description**
  - Detailed explanation of the encountered issue (crash/error/abnormal behavior, etc.).
- **Log file**
  - Path: `<DeadCellsGameRoot>/coremod/logs/log_latest.log`.
- **Additional materials**
  - Screenshots taken when the issue occurred.
  - Other relevant debugging information.
