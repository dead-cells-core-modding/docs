# 通过 Steam 创意工坊安装 DCCM 核心

本教程将指导您完成 Dead Cells Core Modding (DCCM) 核心的安装过程

:::tip

对于 **Linux** ，为 **Dead Cells** 启用 **Proton** 是必须的。请参阅 [链接](https://steamcommunity.com/games/221410/announcements/detail/1696055855739350561)。

:::

## 安装

### 获取 SteamStartShell

从 [Github](https://github.com/dead-cells-core-modding/core/releases/download/v35.8.4/deadcells.exe) 获取 **SteamStartShell**

### 覆盖

1. 找到 游戏根目录。
2. 使用 **SteamStartShell** (`deadcells.exe`) 游戏根目录下的 `deadcells.exe`。

### 启动

通过 **Steam** 启动游戏。应该能看到类似下面的输出：

```txt
[12:27:23 WRN] DCCM is not installed.
[12:27:23 INF] DCCM is downloading...
```

将会从 **Steam 创意工坊** 下载 **DCCM 核心**，请耐心等待。

:::tip
如果该过程持续过久，请确认网络状况并尝试重新启动 **Steam**。
:::

下载完成后会启动游戏。

### 安装 Mods

与 [手动安装](./install-core.md) 不同，**通过 Steam 创意工坊安装 DCCM 核心** 可以访问发布到 **Steam 创意工坊**的  Mods。

关于如何在 **Steam 创意工坊** 订阅 **Mods**, 此处不再阐述。

- 大部分 **Steam 创意工坊** 中的 **DCCM Mods** 的名称以 **[DCCM]** 开头。
